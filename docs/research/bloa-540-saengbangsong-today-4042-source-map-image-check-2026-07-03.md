- `검증 불가` `우리 동네 주치의`는 이번 heartbeat 기준 공개 출처에서 의사 실명, 병원명, 주소를 고정하지 못했다. 의료·개인정보·의료광고 리스크 때문에 독립 발행 소재로 다루기 어렵다.
- `확인됨` `투데이 연예`는 방송 내 고정 정보 코너로만 확인되며, 독립 블로그 글로 전환할 검색 의도와 1차 출처 묶음이 약하다.

## 세그먼트별 메모

### 1. 맛있는 퇴근 - 오늘은 내 맘대로 차리는 한우 식탁

- `확인됨`
  - SBS 4042회 페이지 설명과 AMP 원문에 `맛있는 퇴근ㅣ오늘은 내 맘대로 차리는 한우 식탁`이 직접 적혀 있다.
  - 더쎈뉴스는 2026-07-02 기사에서 서울의 `한우정육소 용두동직영점`을 방송 장소로 명시했다.
  - TopStarNews는 `서울 동대문구 용두동, 용두역과 제기동역 사이` 단서와 `한우 100g`, `오늘 추천 부위` 메뉴 구성을 보강한다.
  - Tableling 공개 페이지 구조화 데이터에 `서울 동대문구 고산자로31길 6`, 좌표 `37.57685123495713, 127.03764885534207`, 영업시간, `한우100g 9,900원`, `2인 오늘 추천부위 49,500원`, `주차가능`이 함께 노출된다.
- `단일 출처`
  - 전화번호는 Tableling 페이지에 비어 있어 이번 heartbeat에서 공식 연락처를 고정하지 못했다.
  - 휴무는 Tableling상 `정기 휴무일 -`로 보이지만, 공식 자사 홈페이지·지도 프로필 재확인은 아직 없다.
- 지도/좌표
  - 주소: `서울 동대문구 고산자로31길 6`
  - 좌표: `37.57685123495713, 127.03764885534207`
  - 지도 링크: `https://maps.google.com/?q=37.57685123495713,127.03764885534207`

  - 다만 Tableling 이미지가 사업자 업로드인지 이용자 리뷰 업로드인지가 섞여 있고, 상업적 재배포 허용 문구는 찾지 못했다.
  - Blogger가 바로 업로드하기보다 `허락 확인 가능 시 사용`, 아니면 본문 상단엔 지도+정보 카드 중심으로 처리하는 쪽이 안전하다.

  - 방송 직후 글감으로는 4042회 중 가장 강하다.

### 2. 오르락내리락 - 동강 물굽이를 내려다보는 산, 백운산!

- `확인됨`
  - SBS 4042회 페이지 설명과 AMP 원문에 `오르락내리락ㅣ동강 물굽이를 내려다보는 산, 백운산!`이 직접 적혀 있다.
  - KTO 계열 관광 데이터 페이지는 `백운산은 정선에서 흘러나온 조양강과 동남천이 합쳐져 형성된 해발 883m의 산`, `동강의 중간 지점`, `점재교 1.1km`, `문희마을-백룡동굴 탐방센터-정상 3.2km`를 설명한다.
  - 같은 페이지에서 위치를 `강원특별자치도 정선군 신동읍 점재길 (신동읍)`으로, 길찾기 좌표를 `37.2803813328817, 128.596134310501`로 제공한다.
- `단일 출처`
  - 이번 heartbeat에서는 정선군 공식 관광 페이지 원문 상세를 추가 확보하지 못했고, 코스 설명은 KTO 계열 데이터 1개에 크게 의존한다.
- 지도/좌표
  - 주소: `강원특별자치도 정선군 신동읍 점재길`
  - 좌표: `37.2803813328817, 128.596134310501`

  - 그러나 관광 포털 계열 이미지의 상업적 재배포 허용 조항을 이번 heartbeat에서 직접 확인하지 못했다.
  - 따라서 `이미지 후보 확보` 수준으로는 충분하지만, 실제 hero 업로드 전에는 별도 사용 조건 확인이 필요하다.
- 추천 방향
  - 맛집보다 검색 의도는 약하지만 `정선 백운산`, `동강 물굽이`, `백운산 코스` 조합으로 local 글 초안은 가능하다.
  - 방문 전 확인 안내를 자연스럽게 넣고, 등산 난이도와 코스 길이만 보수적으로 다루는 편이 맞다.

### 3. 우리 동네 주치의 - 목포를 지키는 내과 의사

- `확인됨`
  - SBS 4042회 페이지 설명과 AMP 원문에서 세그먼트 제목은 확인된다.
  - BNT뉴스는 `전남 목포에서 오랜 시간 지역 주민들의 건강을 돌보고 있는 내과 전문의`라는 일반 설명을 보강한다.
- `검증 불가`
  - 의사 실명, 병원명, 정확한 주소, 공식 병원 채널, 진료과 세부 정보는 공개 출처에서 확인하지 못했다.
  - 방송 캡처나 지역 커뮤니티 추정으로 병원명을 맞추는 방식은 개인정보·명예·의료광고 리스크가 크다.
- 리스크
  - YMYL 주제다.
  - 특정 병원·의사 소개 글은 의료법상 광고성 오해, 실명 노출, 개인정보 처리 이슈가 겹친다.
  - 방송 맥락만으로 건강 정보형 글로 바꾸더라도 병원 특정이 안 된 상태에서는 4042회와의 연결성이 약하다.
- 추천 방향
  - 이번 회차에서는 스킵이 안전하다.
  - 향후 SBS 공식 클립 또는 병원 측 공식 언급이 공개될 때만 재검토한다.
